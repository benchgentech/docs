# Creating Agents

### Ways to Create Agents in Agentspace

1. **Use Templates** (Recommended for beginners)

   - Choose from pre-built agents (E-commerce, Support, Browser, etc.)

   - Customize prompts, tools, and behavior

2. **Build from Scratch**

   - Define goal, tools, and LLM configuration

   - Use no-code builder or code editor

3. **Import Existing Agents**

   - Connect via API endpoint

   - GitHub sync

   - Custom code upload

**Next Step After Creation**

Spin up an Environment → Connect the Agent → Run Benchmark
