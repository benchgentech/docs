# Agent-to-Agent (A2A) Connection Guide

Connect multiple agents so they can collaborate inside the same environment.

### Supported Protocol

- **Server-Sent Events (SSE)** for real-time communication

### Quick Setup

1. Enable A2A mode in Agent Settings

2. Copy the SSE endpoint

3. Configure the second agent to listen to the first agent’s events

**Example Use Cases**

- Supervisor agent \+ worker agents

- Multi-agent research team

- Orchestrator \+ tool agents
